# 🌿 Leaf Age Detection using YOLOv8

## 📌 Project Overview
This project detects and classifies plant leaves into:
- Young Leaves
- Old Leaves

using deep learning-based object detection (YOLOv8).

---

## ⚙️ Technologies Used
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- PyTorch
- Google Colab

---

## 🧠 Model Details
- Model: YOLOv8n
- Task: Object Detection
- Output: Bounding boxes + class labels + confidence scores

---

## 📂 Classes
- 0 → Young Leaf
- 1 → Old Leaf
